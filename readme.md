Jednoduchá virtuální klaviatura vytvořena pomocí Pythonu a knihovny tkinter.

SOUBORY
   main.py              - hlavní soubor, obsahuje GUI (grafické uživatelské rozhraní, které zahrnuje klaviaturu, panely, které zobrazují informace o notách a akordech a tlačítko "Reset")
   logic.py             - soubor, který řeší veškerou hudební teorii, která se skrývá za rozpoznáváním akordů
   chordlib.py          - knihovna akordů, které program zná
   user_guide.md        - uživatelská dokumentace
   programmer_guide.md  - programátorská dokumentace
   readme.md            - tento soubor

JAK SPUSTIT PROGRAM
    spusťte main.py

POŽADAVKY
    mít nainstalovaný Python 3. Tkinter je součástí standardní instalace Pythonu, a není tedy třeba ho zvlášť instalovat.

Pro více informací si přečtěte uživatelskou a programátorskou dokumentaci.
